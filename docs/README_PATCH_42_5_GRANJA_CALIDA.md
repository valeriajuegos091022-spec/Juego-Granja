# README PATCH 42.5 - Granja calida / menos tenebrosa

## Objetivo

Hacer que la pantalla principal se sienta más cálida y más de granja.

El Patch 42.4 mejoró el contraste, pero se sentía demasiado oscuro. Este parche aclara la UI sin volver a colores chillones.

## Qué cambia

### Paleta

- Más madera miel.
- Más crema cálido.
- Más verde campo.
- Dorado suave como acento.
- Menos carbón/negro en la pantalla principal.

### Dashboard

- Fondo general más claro.
- HUD superior con más luz.
- Botones principales más cálidos.
- Panel **¿Qué hacer ahora?** menos oscuro.
- Imagen de la granja con más brillo y vida.
