# Patch 44.23 - Mercado más compacto y categorías escalables

## Objetivo
Compactar un poco más las filas del Mercado y preparar desde ahora una estructura de categorías que evite una lista interminable cuando crezca el catálogo.

## Categorías previstas
- Todos
- Básicos
- Lácteos
- Cárnicos
- Elaborados
- Conservas
- Cultivos

Solo aparecen las categorías que actualmente contienen productos. Las categorías futuras se mostrarán automáticamente cuando se registren productos compatibles.

## Cambios incluidos
- Filas de Mercado ligeramente más pequeñas.
- Nueva barra de filtros por categorías.
- Conteo de productos por categoría.
- Clasificación automática basada en el catálogo central de productos.
- Estado vacío preparado por seguridad.
