## Cambios

- Botones **Volver** y **Menú** mejor centrados y con más ancho.
- Comprar cambia a verde finca.
- Vender cambia a rojo terracota.
- Todo cambia a ámbar/madera.
- Se mantiene fuente legible y botones compactos.
- Cache actualizado a `43_9_mercado_botones_finca`.

## No cambia

- Compra/venta.
- Vender todo.

3. Probar Comprar.
4. Probar Vender.
