# Patch 44.2 — Portada premium real

Este parche rehace la portada principal porque el 44.1 mejoró, pero todavía se sentía demasiado pesada y poco terminada.

- `index.html`
- `css/style.css`
- `docs/CHANGELOG.md`

## Cambios

- Portada replanteada como entrada cálida a la finca.
- Fondo de granja más visible y con menos panel oscuro encima.
- Menú principal más compacto.
- Botones tipo placa premium en crema, oliva y dorado suave.
- Nueva Partida gana jerarquía principal.
- Continuar y Cargar quedan como acciones secundarias.
- Ajustes y Salir quedan más discretos.
- Se elimina el sello visual "FL" del bloque de portada.
- Cache actualizado a `44_2_portada_premium_real`.

## No cambia

- Guardado.
- Economía.
- Mercado.
- Contratos.
- Corrales.
- Maquinarias.
- Lógica de selección de personaje.
- Retratos de granjero/granjera.

## QA recomendado

1. Abrir portada en ventana normal.
2. Abrir portada en pantalla completa.
3. Confirmar que el fondo respira y no se ve tapado por paneles oscuros.
4. Probar Nueva Partida.
5. Probar Continuar y Cargar.
6. Revisar selector de personaje.
7. Confirmar que no hay textos cortados en móvil.
