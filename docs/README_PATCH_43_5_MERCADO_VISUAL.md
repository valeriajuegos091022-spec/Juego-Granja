# Patch 43.5 — Mercado visual premium

Este parche pule visualmente la pantalla de mercado para que combine mejor con el mapa de finca.

## Archivos modificados

- `index.html`
- `css/style.css`
- `docs/CHANGELOG.md`
- `docs/HANDOFF.md`
- `docs/TODO.md`
- `docs/UI_DESIGN_BIBLE.md`
- `docs/BUGS.md`
- `docs/README_PATCH_43_5_MERCADO_VISUAL.md`

## Cambios visuales

- Mercado con fondo y paneles más cálidos.
- Topbar organizada en navegación, título y recursos.
- Título cambiado a **Mercado de la Finca**.
- Tarjetas de resumen más claras.
- Lista de productos con tarjetas crema/madera.
- Botones Comprar/Vender con verde agrícola y madera cálida.
- Popup de producto más claro y legible.

## No cambia

- Economía.
- Precios.
- Demanda.
- Compra/venta.
- Guardado.
- Balance.
- Datos de partidas.

## Prueba recomendada

1. Abrir una partida.
2. Entrar al Mercado desde el mapa.
3. Revisar que dinero y día se vean bien.
4. Comprar 1 producto.
5. Vender 1 producto.
6. Abrir un producto para revisar demanda y gráfica.
7. Volver al mapa.
