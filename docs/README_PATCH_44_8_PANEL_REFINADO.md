# Patch 44.8 - Panel refinado

## Objetivo
Refinar el Patch 44.7, que fue el mejor punto de partida reciente para la portada principal.

## Cambios incluidos

- Se agranda un poco el conjunto para que tenga más presencia en pantalla.

- Se cambia la paleta hacia crema envejecida, nogal oscuro y oliva suave.
- Se suaviza el exceso de naranja y se busca un acabado menos plástico.
