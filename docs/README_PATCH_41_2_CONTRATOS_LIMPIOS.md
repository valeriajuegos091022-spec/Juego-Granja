# PATCH 41.2 — Contratos más limpios

Este parche limpia visualmente las tarjetas de contratos.

## Cambios

- Se oculta la línea técnica/económica de los contratos.
- Ya no se muestra el cálculo de mercado ni ganancia estimada.
- Las tarjetas quedan más limpias para que el jugador descubra la estrategia por sí mismo.
- Se mantienen recompensa, plazo, requisitos y botones.
- No cambia la lógica de contratos, mercado ni guardado.

- index.html
- js/modules/contracts.js
- css/style.css
