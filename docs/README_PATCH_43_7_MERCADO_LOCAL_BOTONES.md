# Patch 43.7 — Mercado local + botones vivos

Este parche ajusta detalles visuales del mercado después de probar el Patch 43.6.

## Archivos modificados

- `index.html`
- `css/style.css`
- `docs/CHANGELOG.md`
- `docs/HANDOFF.md`
- `docs/TODO.md`
- `docs/UI_DESIGN_BIBLE.md`
- `docs/BUGS.md`
- `docs/README_PATCH_43_7_MERCADO_LOCAL_BOTONES.md`

## Cambios

- El título visible ahora es **Mercado local**.
- Comprar usa un verde más vivo.
- Vender usa un rojo/naranja más vivo.
- Comprar/Vender reciben iconos visuales por CSS.
- Cache actualizado a `43_7_mercado_local_botones`.

## No cambia

- Economía.
- Precios.
- Demanda.
- Compra/venta.
- Guardado.
- Balance.
- Datos de partidas.

## Prueba recomendada

1. Abrir una partida.
2. Entrar al Mercado.
3. Confirmar que el título dice **Mercado local**.
4. Comprar 1 producto desde la lista.
5. Vender 1 producto desde la lista.
6. Abrir un producto y probar Comprar/Vender desde el popup.
