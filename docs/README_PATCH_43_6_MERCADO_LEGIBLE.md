# Patch 43.6 — Mercado legible + tono mapa

Este parche corrige la primera versión visual del mercado 43.5 después de verla en pantalla real.

## Archivos modificados

- `index.html`
- `css/style.css`
- `docs/CHANGELOG.md`
- `docs/HANDOFF.md`
- `docs/TODO.md`
- `docs/UI_DESIGN_BIBLE.md`
- `docs/BUGS.md`
- `docs/README_PATCH_43_6_MERCADO_LEGIBLE.md`

## Cambios

- Mercado más cercano a la pantalla principal.
- Marco principal en madera oscura.
- Headers y botones con verde agrícola.
- Crema solo en tarjetas internas, no como fondo dominante.
- Columnas más anchas para que los textos no se vean corridos.
- Mejor lectura en producto, precio, tienes y comprar/vender.
- Cache actualizado a `43_6_mercado_legible`.

## No cambia

- Economía.
- Precios.
- Demanda.
- Compra/venta.
- Guardado.
- Balance.
- Datos de partidas.

## Prueba recomendada

1. Abrir una partida.
2. Ir al Mercado.
3. Revisar si sigue la tónica de la pantalla principal.
4. Confirmar que textos y botones se leen bien.
5. Comprar 1 producto.
6. Vender 1 producto.
7. Abrir un producto y revisar el popup.
