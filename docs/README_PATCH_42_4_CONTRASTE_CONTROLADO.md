# README PATCH 42.4 - Contraste controlado

## Objetivo

Corregir que la pantalla principal quedara demasiado marrón/carmelita después del Patch 42.3.

El objetivo no es volver a muchos colores, sino agregar contraste sobrio.

## Qué cambia

### Paleta

- Se mantiene madera oscura como base.
- Se agregan tonos carbón para profundidad.
- Se agrega verde oliva profundo para separar zonas.
- El dorado queda como acento suave.

### Dashboard

- HUD superior con más diferencia entre dinero, día, almacén y guardado.
- Botones principales con más separación visual, sin usar colores chillones.
- Panel **¿Qué hacer ahora?** menos plano y menos marrón.
- Tarjeta de la granja con acentos más oliva.
