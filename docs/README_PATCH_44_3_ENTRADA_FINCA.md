## Cambios

- Se agrega un letrero grande de madera para **Farm Legacy**.
- El menú principal pasa a placas verticales colgadas.
- Se elimina la sensación de cajas flotantes del 44.2.
- Se mantiene el fondo existente del juego.
- Botones sin emojis ni iconos baratos.
- Cache actualizado a `44_3_entrada_finca`.

## QA recomendado

1. Abrir portada en PC.
2. Confirmar que el letrero se ve como portada real.
3. Confirmar que los botones se leen bien.
4. Probar Nueva Partida.
5. Probar Continuar y Cargar Partida.
6. Revisar pantalla baja y móvil.
